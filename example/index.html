A architecture model example

This example is stolen from the C4 Model.

Internet Banking System

[Software System]

Allows customers to view information about their bank accounts, and make payments.

Gets account information from, and makes payments using MBS.

Sends e-mail using ES.

Personal Banking Customer

[Person]

Allows customers to view information about their bank accounts, and make payments.

Views account balances, and makes payments using IBS.

Mainframe Banking System

[Software System]

Stores all the core banking information about customers, accounts, transactions, etc.

E-Mail System

[Software System]

The internal Microsoft Exchange e-mail system.

Sends e-mail to PBC.
